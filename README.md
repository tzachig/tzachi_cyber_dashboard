# Hello World Confetti App 🎉

A beautiful hello world application with confetti animation celebration!

## Features

✨ **Confetti Animation** - Colorful animated confetti particles
🎨 **React Component** - Modern React with hooks
⚙️ **Express Backend** - Simple Node.js backend
🚀 **Ready for Vercel** - Deploy with one click
🎯 **Fully Responsive** - Works on all devices

## Getting Started

### Install Dependencies
```bash
cd hello-world-confetti
npm install
```

### Run Locally
```bash
npm run dev
```

Visit `http://localhost:3000` in your browser.

## Project Structure

```
hello-world-confetti/
```

## API Endpoints

- `GET /api/hello` - Returns hello message
- `GET /api/confetti` - Returns confetti config
- `GET /health` - Health check

## Deploy to Vercel

1. Push to GitHub:
```bash
git add .
git commit -m "Initial commit: Hello World with Confetti"
git push origin main
```

2. Go to [Vercel](https://vercel.com)
3. Import your GitHub repo
4. Deploy!

## Technologies

- **Frontend**: React 18 + Tailwind CSS
- **Backend**: Express.js
- **Deployment**: Vercel
